# Reminisce.me

## Installation

First clone the repository somewhere:

    $ git clone git@github.com:reminisceme/app.git

Open two or more panes/tabs/windows.

In the first pane:

    $ cd app/react
    $ npm install
    $ make watch

In the second pane:

    $ cd app/meteor
    $ cp path/to/settings.json ./
    $ meteor --settings settings.json

In a third pane:

    open http://local.reminisce.me
